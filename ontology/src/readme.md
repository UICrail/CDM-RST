Folder containing the Rolling Stock ontologies. Versions are in subfolders.

| file   | purpose |
| ------ | ------- |
| cons.ttl | train consist : vehicles and formations as sequences of oriented vehicles or other formations |
| typo.ttl | rolling stock typology ("locomotive", "wagon"...) |

rst.ttl merely imports the latest versions of all other ontologies; it is not versioned. Use with care.
